## Setup Instructions

### 1. Basic Setup
```bash
# Clone or download the files
# Ensure all files are in the same directory:
# - index.html
# - script.js
# - styles.css
# - CHAKRA_README.md
```

### 2. AdSense Integration

2. **Replace Placeholder Codes**: In `index.html`, replace:
   ```html
   <!-- Replace ca-pub-XXXXXXXXXXXXXXXX with your AdSense publisher ID -->
   <ins class="adsbygoogle"
        style="display:block"
        data-ad-client="ca-pub-YOUR-PUBLISHER-ID"
        data-ad-slot="YOUR-AD-SLOT-ID"
        data-ad-format="auto"
        data-full-width-responsive="true"></ins>
   ```
3. **Add AdSense Script**: Add this to your HTML head:
   ```html
   <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-YOUR-PUBLISHER-ID"
           crossorigin="anonymous"></script>
   ```

### 3. Audio Files (Optional)
For the meditation player, add audio files to an `audio/` directory:
```
audio/
  root-meditation.mp3
  sacral-meditation.mp3
  solar-meditation.mp3
  heart-meditation.mp3
  throat-meditation.mp3
  third-eye-meditation.mp3
  crown-meditation.mp3
  full-chakra-meditation.mp3
```

### 4. Customization
- **Colors**: Modify CSS variables in `styles.css`
- **Content**: Update chakra descriptions in `index.html`
- **Images**: Add your own chakra graphics
- **Branding**: Update logo and company name
